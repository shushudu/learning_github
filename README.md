# ssd_show_big_files

Выводит список самых больших файлов в заданном расположении

```
ssd_show_big_files <source> <max_count>
```

## Синтаксис командной строки

| Нотация                        | Описание
|--------------------------------|---------
| ```text```                     | Текст без скобок - его необходимо вводить в точности так как показано
| ```<text>```                   | Текст в угловых скобок - вместо него нужно ввести конкретное значение
| ```[text]```                   | Текст в квадратных скобках - необязательные элементы, можно пропустить
| ```{text}```                   | Текст в фигурных скобках - набор обязательных элементов. Нужно выбрать один
| ```Вертикальная полоса (\|)``` | Разделитель для взаимоисключающих элементов. Нужно выбрать один
| ```Многоточие (…)```           | Элементы, которые могут повторяться и использоваться несколько раз


## Параметры командной строки

| Параметр          | Описание
|-------------------|---------
| ```--help```      | Показать справку
| ```--version```   | Показать версию утилиты
| ```<source>```    | Указание пути к исходному каталогу. Путь должен завершаться символом ```\\```
| ```<max_count>``` | Максимальное количество выводимых строк, целое число больше нуля


## Коды выхода (return)

| Значение | Описание
|----------|---------
| 0        | Сбой не обнаружен
| 1        | Ошибка

## Примеры использования

Например, чтобы вывести список из 10 самых больших файлов на диске c:\, введите:
```
ssd_show_big_files c:\ 10
```

Показать справку по командам:
```
ssd_show_big_files --help
```

Показать версию утилиты:
```
ssd_show_big_files --version
```
